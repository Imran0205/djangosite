Шаг 1. Начало

В начале я рекомендую вам изучить язык программирования Python. Он помогает понимать базовую концепцию программирования и он лёгкий в изучении. На его основе вы сможете создавать базовые сайты программы и игры. Вам может показаться что он не так хорош для больших проектов но у него есть и свои плюсы. Он очень хорошо работает с ИИ из-за чего его можно использовать в больших проектах. Насчёт его минусов то он является немного медленным по сравнению с другими языками. При создании сайтов этого особо не видно но в больших проектах эта разница заметна.

Шаг 2. Первые проекты

После изучения языка Python вы можете развиваться дальше в сторону Python или изучать другие языки. Я советую вам изучать дальше в сторону Python потому что потом можно будет не заморачиваться по этому поводу и спокойно интегрировать его в свои проекты. При выполнении первого шага у вас есть несколько вариантов направления. Конечно вы в итоге выучите их все, но откуда можно начать? Можно создавать сайты с помощью Django или Flask, можно создавать программы с помощью PyQt5 или изучить библиотеку FastAPI. Вы также можете выучить работу с ИИ, чатом GPT и телеграм-ботами. Сначала я советую изучить работу с сайтами ведь она очень простая и полезная. Потом же можно изучать всё остальное на выбор. Ниже вы можете посмотреть требования к изучениям ↓.
Создание сайтов на Django Для создания сайтов на Django вам потребуется знать язык программирования Python, языки разметки HTML и CSS. Языки разметки HTML и CSS обезательны для создания сайтов на любой программе и с любым языком. HTML нужен для создания обьектов на сайте а CSS для добавления стиля к обьектам. Язык программирования нужен для добавления функциональной части к сайту.
